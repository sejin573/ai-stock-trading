# 뉴스 기반 한국 주식 자동 포트폴리오 대시보드

## Slide 1. 프로젝트 소개
- 네이버 뉴스, KIS Open API, pykrx를 결합한 한국 주식 포트폴리오 대시보드
- 실시간 가격, 뉴스 흐름, 후보 종목 탐색, 모의 자동매매를 한 화면에서 모니터링

## Slide 2. 문제 정의
- 뉴스와 주가를 따로 보느라 종목 판단 속도가 느림
- 단기 기대 수익이 높은 종목을 빠르게 골라내기 어려움
- 포지션 수익률, 평가손익, 매매 이력이 분산되어 관리 피로가 큼

## Slide 3. 해결 방식
- 뉴스: 네이버 뉴스 검색 API
- 시세 및 주문: KIS Open API
- 시장 보조 데이터 및 백테스트: pykrx
- UI: Streamlit 대시보드

## Slide 4. 핵심 기능
- 한국 상장사 전체 목록 기반 종목 선택
- 3초 주기 자동 갱신
- 변동성/예상 상승률 기반 상승 기대주 추천
- 모의투자 기반 자동 매수/자동 매도
- 최근 자동 매매 이력, 보유 수량, 총 투자원금, 총 평가금액, 누적 수익률 표시

## Slide 5. 자동매매 로직
- 시장 후보군 스캔
- 뉴스 감성, 가격 모멘텀, 변동성, 기대수익률을 종합 점수화
- 기준을 넘는 종목을 자동 매수 후보로 채택
- 목표 수익률 도달 시 자동 매도
- 거래 결과를 기반으로 가중치를 조정하는 학습 루프 반영

## Slide 6. UI 개선 포인트
- 보유 종목별 총 매수금액, 현재 평가금액, 평가손익 표시
- 평가손익 및 수익률 색상 강조
- 수익률 상위 / 손실률 상위 / 평가금액 상위 정렬 토글 제공
- 대시보드형 요약 카드로 포트폴리오 상태를 빠르게 파악

## Slide 7. 기대 효과 및 확장
- 단기 기회 종목 탐색 시간 단축
- 뉴스 이벤트와 가격 반응을 동시에 관찰 가능
- 실전 전 모의투자로 전략 검증 가능
- 향후 실거래 연계, 모델 고도화, 리스크 관리 기능으로 확장 가능
